# Housman3
Authorship Attribution Classifier with GUI 

Housman upload for DMU Blog Posts assessment.
Experiment was carried out using SHAXICAN Training Data (https://gabrielegan.com/shaxican/)
Settings used were:
- 350 MFW
- 1 Chunks
- 50% Culling

Classification Results:
Horatio - Male
Laertes - Male
Claudius - Male
Hamlet - Male
Ophelia - Female
Gertrude - Male
